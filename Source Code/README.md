# Source Code

This folder contains the complete implementation of the Academic Analytics System developed using R Shiny.

The source code includes:

* Login System
* Student Dashboard
* Parent Dashboard
* Doctor Dashboard
* Dean/Admin Dashboard
* Data Processing Functions
* Analytics Components
* Visualization Modules
* Attendance Monitoring Logic

Main Files:

* app.R
* ui_login.R
* ui_admin.R
* doctor_dashboard.R
* parent_dashboard.R
* student_dashboard_only.R
* admin_server.R

The system uses interactive dashboards, dynamic charts, statistical analysis, and role-based access control to provide personalized educational insights.
