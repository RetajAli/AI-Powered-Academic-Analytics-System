# Source Code

## Overview

This folder contains the implementation files of the AI-Powered Academic Analytics System.

The source code combines Artificial Intelligence, Data Analytics, Dashboard Development, Mobile Application Development, and Database Design into a single educational analytics platform.

## Contents

### Python Modules

AI and Computer Vision components:

* main.py
* emotion_detection_module.py
* face_recognition_module.py
* sign_language_module.py
* csv_logger.py

These modules are responsible for:

* Face recognition
* Attendance tracking
* Emotion detection
* Sign language recognition
* Data logging

### R Dashboards

Interactive dashboards developed using R Shiny:

* app.R
* admin_server.R
* doctor_dashboard.R
* parent_dashboard.R
* student_dashboard_only.R

Features include:

* Student performance visualization
* Parent monitoring dashboards
* Instructor analytics
* Administrative reporting

### Flutter Mobile Application

Mobile application source files:

* lib.zip
* pubspec.lock
* analysis_options.yaml

The application provides a mobile interface for accessing academic analytics and educational services.

### Database

* Schema.sql

Contains the database schema used to store and manage system data.

### Datasets

* StudentPicsDataset.csv
* detections_log.csv

Used for training, testing, and logging AI-related operations.

### MediaPipe Models

* face_landmarker.task
* hand_landmarker.task

Pretrained models used for computer vision and gesture recognition tasks.

## Technologies

* Python
* OpenCV
* MediaPipe
* R Shiny
* Flutter
* Dart
* SQL

## Academic Purpose

This project was developed as part of the Advanced Statistics course to demonstrate the integration of statistical analysis, artificial intelligence, computer vision, and educational analytics in a practical system.
