# AI-Powered Academic Analytics System

## Overview

The AI-Powered Academic Analytics System is an intelligent educational platform developed to monitor, analyze, and improve student academic performance through interactive dashboards and artificial intelligence technologies.

The system provides multiple dashboards for students, parents, instructors, and administrators, allowing each user to access personalized academic insights and performance analytics.

In addition to academic monitoring, the platform integrates AI-powered classroom analytics using computer vision techniques to track attendance, engagement, participation, and student activity during lecture sessions.

## Features

* AI-powered student performance analysis
* Student dashboard for academic monitoring
* Parent dashboard for student progress tracking
* Instructor dashboard for classroom analytics
* Administrator dashboard for system management
* Facial recognition for attendance tracking
* Emotion detection during lectures
* Sign language recognition module
* Academic data visualization and reporting
* Interactive dashboards developed with R Shiny
* Mobile application prototype developed with Flutter
