# AI-Powered Academic Analytics System

## Overview

The AI-Powered Academic Analytics System is an intelligent educational platform designed to monitor, analyze, and improve student academic performance through interactive dashboards and artificial intelligence technologies.

The system provides different dashboards for students, parents, instructors, and administrators, allowing each user to access personalized academic insights and performance analytics.

In addition to academic monitoring, the platform integrates AI-powered classroom analytics using computer vision technologies to track attendance, engagement, participation, and student activity during lecture sessions.

## System Architecture

The platform consists of four integrated modules:

1. Student Portal
2. Parent Portal
3. Doctor Portal
4. Dean/Admin Portal

Each module accesses the same academic database while providing customized analytics and visualizations based on user roles.
